# Is Leap Year

This indicates whether the Day.js object's year is a leap year or not.

<Callout>
  This requires the [IsLeapYear](../plugin/is-leap-year) plugin to work
</Callout>

```js
dayjs.extend(isLeapYear);

dayjs("2000-01-01").isLeapYear(); // true
```
