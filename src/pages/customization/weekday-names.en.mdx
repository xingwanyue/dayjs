# Weekday Names

`Locale#weekdays` should be an array of the weekdays names.

<Callout>
  This requires the [UpdateLocale](../plugin/update-locale) plugin to work
</Callout>

```js
dayjs.extend(updateLocale);

dayjs.updateLocale("en", {
  weekdays: [
    "Sunday",
    "Monday",
    "Tuesday",
    "Wednesday",
    "Thursday",
    "Friday",
    "Saturday",
  ],
});
```
