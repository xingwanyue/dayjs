# Browser

```js
<script src="path/to/dayjs/dayjs.min.js"></script>
<script>
  dayjs().format()
</script>
```

#### CDN resource

<Callout>
  Day.js can be included by way of a CDN provider like cdnjs.com, unpkg and
  jsDelivr ...
</Callout>

```js
<!-- CDN example (jsDelivr) -->
<script src="https://cdn.jsdelivr.net/npm/dayjs@1/dayjs.min.js"></script>
<script>dayjs().format()</script>
```

Check here for more information about loading [locale](../i18n/loading-into-browser) and [plugin](../plugin/loading-into-browser).
